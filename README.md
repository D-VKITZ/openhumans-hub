# OpenHumans Hub

OpenHumans Hub — Citizen Science Daten-Explorer.

## Features

- Browse and explore OpenHumans public datasets
- Data visualization and statistical analysis
- OpenHumans API integration for data upload
- Project discovery and contribution tracking
- Data export in multiple formats

## Tech Stack

JavaScript ES6+ · HTML5 Canvas · OpenHumans API

## Quick Start

```bash
git clone https://github.com/D-VKITZ/openhumans-hub.git
cd openhumans-hub
npm install
npm run dev
```
